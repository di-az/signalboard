# Signalboard
A modular Go platform for aggregating personal data sources and rendering them to different outputs such as LED displays, web dashboards, and other clients.

## Overview
Signalboard collects information from various sources and turns it into frontend-ready consumable information.

The long-term goal is to support multiple independent data sources and multiple presentation targets, allowing the same information to be displayed on an LED matrix, a web interface, desktop widgets, and more.
